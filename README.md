# Hello Clojure CLR

A simple "Hello World" program in Clojure CLR

### Install Clojure CLR

    # Clojure.Main
    dotnet tool install --global Clojure.Main --version 1.12.0-alpha10

    # Clojure.Cljr
    dotnet tool install --global Clojure.Cljr --version 0.1.0-alpha2

### Run

    cljr -M -m hello.main
